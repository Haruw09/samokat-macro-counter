# Samokat Macro Counter

Неофициальный инструмент для расчёта КБЖУ на основе текущей корзины Samokat.

## Зачем это нужно

Этот проект помогает ответить на вопрос:

**Если я съем эту корзину, сколько КБЖУ в день я получу и сколько мне останется добрать?**

## Как это работает

1. Пользователь задаёт дневную норму:
   - калории
   - белки
   - жиры
   - углеводы

2. Программа считывает корзину Samokat

3. Для каждого товара задаётся:
   - на сколько дней он рассчитан
   - способ учёта:
     - упаковка
     - граммы
     - штуки
     - столовые ложки

4. При необходимости вводятся:
   - граммы / штуки / ложки
   - средний вес

5. Программа считает:
   - среднесуточный вклад КБЖУ
   - общий итог
   - остаток до нормы

## Дисклеймер

Этот проект:

- не является официальным продуктом Samokat
- не аффилирован с Samokat
- создан для личного использования
- не предназначен для массового сбора данных
- не гарантирует точность КБЖУ

Все товарные знаки принадлежат их правообладателям.
